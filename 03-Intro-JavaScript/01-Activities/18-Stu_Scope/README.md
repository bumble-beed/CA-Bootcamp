# Loop Practice

## Objective
Practice for and while.

## Steps
1. Print numbers 1..10 using for.
2. Print 5..1 using while.
3. Add a safety guard to avoid infinite loops.

## Done When
- Correct order and counts.
- No infinite loops.

## Tips
- Place increments correctly; use a max-iteration guard.
